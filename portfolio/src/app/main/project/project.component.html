  • Docker – Containerization of microservices for seamless deployment
  • Java (Spring Boot) – Powerful backend for scalable performance
  • Angular – Modern, dynamic front-end for enhanced user experience
  • Tailwind CSS – Elegant, responsive design
  • UML – Architecture modeling for clear system structure
  • Jenkins – Automated CI/CD pipelines for continuous delivery
  • Figma – Sleek UI/UX design for modern look
  • Microservices – Scalable and independent services architecture
  • Keycloak – Secure user authentication and authorization
  • OAuth – User authentication and secure data access

Project Goals

  • Provide a seamless, user-friendly interface for booking management
  • Develop a scalable backend using microservices for future expansion

  • Create a visually appealing UI with Figma and responsive design with Tailwind CSS
